🧭 ALADDIN Mobile Navigation System

Адаптивная система навигации для мобильных и десктопных устройств

🎛️ Управление навигацией

📱 Мобильная навигация

Нижняя панель навигации с иконками и подписями. Адаптируется под размер экрана.

🖥️ Десктопная навигация

Горизонтальная панель навигации для больших экранов. Показывается автоматически.

🎨 Особенности дизайна

• Glassmorphism эффекты с размытием
• Плавные анимации переходов
• Haptic feedback при касании
• Адаптивная типографика
• Storm Sky цветовая схема

💻 Пример кода (Swift)

// Создание навигации
let navigationItems = NavigationFactory.createALADDINNavigation()
let navigationController = MobileNavigationController(
    navigationItems: navigationItems,
    contentViewController: contentVC
)

// Настройка делегата
navigationController.delegate = self

// Обновление badge
navigationController.updateBadge(for: "security", badge: "3")

// Переключение выбранного элемента
navigationController.setSelectedIndex(2, animated: true)

💻 Пример кода (Kotlin)

// Создание навигации
val navigationItems = NavigationFactory.createALADDINNavigation()
val navigationController = MobileNavigationController(context, fragmentManager)
navigationController.setup(containerView, navigationItems, contentFragment)

// Настройка делегата
navigationController.setDelegate(this)

// Обновление badge
navigationController.updateBadge("security", "3")

// Переключение выбранного элемента
navigationController.setSelectedIndex(2, true)
