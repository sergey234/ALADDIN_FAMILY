🎨 ALADDIN Glassmorphism Effects

Современные стеклянные эффекты для мобильного приложения безопасности

🎛️ Интерактивная панель управления

🔮 Light Glass Effect

Легкий стеклянный эффект с минимальным размытием и прозрачностью. Идеально подходит для тонких элементов интерфейса.

💎 Medium Glass Effect

Средний стеклянный эффект с балансом между прозрачностью и читаемостью. Рекомендуется для основных элементов.

🏔️ Heavy Glass Effect

Сильный стеклянный эффект с максимальным размытием. Создает впечатление толстого стекла.

❄️ Frosted Glass Effect

Морозное стекло с высокой прозрачностью и размытием. Создает эффект матового стекла.

💎 Crystal Clear Effect

Кристально чистое стекло с минимальной прозрачностью. Максимальная читаемость контента.

⛈️ Storm Sky Effect

Специальный эффект в стиле "Грозовое небо" с золотыми акцентами. Фирменный стиль ALADDIN.

📱 Мобильные элементы

Примеры glassmorphism эффектов для мобильного приложения ALADDIN:

🎨 Storm Sky Theme

Специальная тема "Грозовое небо с золотыми акцентами" для ALADDIN:

Light Storm Sky - Легкий эффект для тонких элементов
Medium Storm Sky - Средний эффект для основных элементов
Heavy Storm Sky - Сильный эффект для акцентных элементов

💻 Пример кода (Swift)

// Создание glassmorphism кнопки
let button = GlassmorphismButton(config: .stormSkyMedium)
button.setTitle("ALADDIN Security", for: .normal)

// Создание glassmorphism карточки
let card = GlassmorphismCard(config: .stormSkyLight)
card.addContent(contentView)

// Применение эффекта к существующему view
glassmorphismView.applyGlassmorphism(config: .stormSkyHeavy)

💻 Пример кода (Kotlin)

// Создание glassmorphism кнопки
val button = GlassmorphismButton(context, GlassmorphismConfig.STORM_SKY_MEDIUM)
button.text = "ALADDIN Security"

// Создание glassmorphism карточки
val card = GlassmorphismCard(context, GlassmorphismConfig.STORM_SKY_LIGHT)
card.addContent(contentView)

// Применение эффекта к существующему view
view.applyGlassmorphism(GlassmorphismConfig.STORM_SKY_HEAVY)
